# Employee Management System | Java AWT + MySQL

A desktop-based Employee Management System built using **Java AWT, JDBC, and MySQL**.  
This system allows users to easily **add, update, view, and delete employee details** through a simple graphical user interface. Suitable for beginners learning Java GUI + database connectivity.
